## Data types And Variables

#### Data types in Java

* Primitive
    * Non - Numeric
        * char
        * boolean `true, false`
    * Numeric
        * Integer
            * byte
            * short
            * int `1, 2, 100, 1001, -1, -3, -105`
            * long
        * Floating point
            * float `1.33, 3.2, 23.323, -43.23, -3.232`
            * double `4.12421395742321E-324`
* Non-Primitive
    * class 
    * interface
    * Array
    * String `"Hello, world!", 'My world!'`
